Результаты базового анализа (Этап 2)

Мы успешно реализовали пайплайн обработки микроскопических снимков для формирования нашей демо-выборки и извлекли базовые признаки с помощью ResNet50.

1. Подготовка данных

  • Исходный датасет: 215 TIFF изображений (55 различных групп текстур).
  • Изображения были нарезаны на тайлы размером 256x256 пикселей без перекрытия (шаг 256 пикселей).
  • Получена отфильтрованная контрольная выборка из 27 439 тайлов (пустые фоновые тайлы исключены).

2. Извлечение признаков (Эмбеддингов)

  • Использована предобученная модель ResNet50 (без дополнительного дообучения).
  • На экстракцию 27 тысяч признаков на CPU было затрачено около 3.5 часов.
  • Результат: Векторы признаков размерностью 2048 для каждого тайла.

3. Визуализация (UMAP) и Метрики качества кластеризации

Мы рассчитали формальные расстояния (метрики "скученности" текстур внутри одной группы) в полученном пространстве эмбеддингов: * Silhouette Score (Cosine): -0.1267 (из диапазона от -1 до 1) * Calinski-Harabasz Index: 122.8 * Davies-Bouldin Index: 4.64

Отрицательный Silhouette Score говорит о сильном нахлесте и пересечении кластеров разных поверхностей и масштабов между собой (векторы смешиваются). Это ожидаемый результат для базовой нейросети, которая не обучена быть инвариантной к изменениям масштаба под микроскопом.

4. Эксперимент: Инвариантность к масштабу (Этап 3)

Для решения проблемы с разделением текстур одного материала из-за разного увеличения микроскопа мы реализовали алгоритм Multi-Scale Feature Fusion. Был проведен эксперимент на сбалансированной подвыборке из 550 тайлов:

  1. Базовый подход (Single-Scale): Silhouette Score: -0.030
  2. Max-Pooling Fusion (0.7x, 1x, 1.5x): Silhouette Score: -0.035

[!NOTE] Вывод для диплома: Применение Max-Pooling "в лоб" для векторов признаков из ImageNet ResNet50 не дало прироста инвариантности к масштабу. Нейросеть не обучалась воспринимать одну и ту же "текстуру" микромира в разных масштабах как один класс. Это сильный аргумент для диплома: он обосновывает, почему базовые CV-эвристики не работают на СЭМ-снимках из коробки и почему в современной литературе (deep clustering) переходят к контрастивному дообучению (Contrastive Learning) вроде SimCLR или BYOL под специфичные задачи микроскопии.
